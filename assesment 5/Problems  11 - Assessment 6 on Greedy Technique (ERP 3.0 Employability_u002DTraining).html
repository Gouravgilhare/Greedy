Problem statement 4

Shyam and Mohan are playing a game where each player has to consume chocolates. Given an array A of size N where Ai represents the quantity of chocolate at ith position. But there is one condition, each player can consume a specific quantity of chocolates only once. If Shyam consumes more amount of chocolate the Mohan then he wins otherwise Mohan wins.

Assume Shyam goes first then find the winner of the game.

Rules of the game:

  • Shyam goes first, followed by Mohan.

  • Each player can consume one chocolate each.

  • A chocolate (element) can be chosen only once.

  • The player who consumes the larger amount of chocolate wins.

  • If both consume the same amount, Mohan wins.

Input format

  • The first line contains an integer N the number of chocolates.

  • The second line contains N space-separated integers representing the array A.

Output format

  • Print Shyam if Shyam wins, otherwise print Mohan.

Sample input 
4

5 5 3 2
Sample output 
Mohan
Explanation

  • Shyam picks the largest chocolate first: 5

  • Mohan picks the next largest, also : 5

  • Since both picked the same amount, Mohan wins.
